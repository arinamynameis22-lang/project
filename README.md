# Система управления продажами автомобилей

Автоматизированная система для учёта автомобилей, отслеживания поступлений, перемещений и продаж, а также формирования отчётов. Предназначена для автосалонов, дилерских центров и компаний по продаже новых и подержанных автомобилей.

## Технологический стек

- **Backend:** Python 3.11+ с FastAPI
- **База данных:** SQLite (с возможностью миграции на PostgreSQL)
- **ORM:** SQLAlchemy
- **Валидация:** Pydantic
- **Frontend:** HTML, CSS, JavaScript

## Структура проекта

```
Progect/
├── backend/           # Backend-приложение (FastAPI, модели, API)
├── frontend/          # Статические файлы интерфейса
├── data/              # Входные файлы (CSV: поступления, перемещения, продажи)
├── tests/             # Тесты
├── requirements.txt   # Зависимости Python
└── README.md
```

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск сервера (после настройки приложения)

```bash
uvicorn backend.main:app --reload
```

## Основные возможности

- Учёт автомобилей на складе (VIN, модель, цвет, цены, статус)
- Загрузка данных из текстовых файлов (CSV)
- Регистрация операций: поступление, перемещение, продажа
- Отслеживание истории перемещений
- Формирование отчётов по остаткам и продажам

## Требования

- Python 3.11 или выше
